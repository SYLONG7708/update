申悅車機助手 APK 更新 圖片位置

APK 更新與圖片位置零基礎教學

這份教學只回答三件事：新版助手 APK 在哪裡、要上傳給車機更新的 APK 放哪裡、每個 App 的兩張介紹圖片放哪裡。

一、最新申悅車機助手 APK 在哪裡

這是要安裝到安卓車機的新版申悅車機助手 APK：

C:\Users\Administrator\Desktop\CODEX 專案資料夾\製作過的 APP 以及資料\申悅車機助手_更新中心_20260603\releases\shen-yue-car-assistant-update-center-v2.0.5.apk

線上下載位置：

https://github.com/SYLONG7708/shen-yue-iphone-assistant/releases/download/assistant-apk/shen-yue-car-assistant-update-center-v2.0.5.apk

二、要上傳更新的 APK 放哪裡

你要讓客戶車機下載的 APK，先放進這個資料夾：

C:\Users\Administrator\Downloads\更新
  1. 把新版 APK 複製到上面資料夾。
  2. 不要把申悅車機助手本體放進更新清單；系統目前會自動排除 助手.apk。
  3. 如果要更新同一個 App，新 APK 的版本碼要比舊 APK 高。

三、產生雲端上傳檔案

把 APK 放好後，到專案資料夾執行：

cd "C:\Users\Administrator\Desktop\CODEX 專案資料夾\製作過的 APP 以及資料\申悅車機助手_更新中心_20260603"
powershell -ExecutionPolicy Bypass -File .\tools\import-update-apks.ps1

執行後會自動產生：

  • APK 雲端檔案：cloud-upload\apks
  • 每個 App 的兩張圖片：cloud-upload\assets\update-gallery
  • 更新清單：cloud-upload\updates.json

五、不要改檔名

替換圖片時，檔名要完全一樣。檔名改掉後，更新中心會找不到圖片。

  1. 準備你的新圖片。
  2. 把新圖片改成表格裡的同名檔案。
  3. 複製到 cloud-upload\assets\update-gallery 覆蓋。
  4. 重新上傳 GitHub。

六、上傳到 GitHub

APK 與圖片都準備好後，執行：

cd "C:\Users\Administrator\Desktop\CODEX 專案資料夾\製作過的 APP 以及資料\申悅車機助手_更新中心_20260603"
powershell -ExecutionPolicy Bypass -File .\tools\upload-github-cloud.ps1

它會自動上傳：

  • 更新清單與圖片到：SYLONG7708/update
  • APK 大檔到：apk-cloud Release
  • 申悅助手網頁到：shen-yue-iphone-assistant

七、上傳後車機怎麼看

  1. 打開申悅車機助手。
  2. 點上方 更新。
  3. 按 重新整理。
  4. 點 App 項目進入單獨頁面。
  5. 你替換的兩張圖片會顯示在單獨頁面內。
